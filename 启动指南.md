# 🌐 网站访问详细指南

## 📋 前置准备

### 1. 确认Python已安装
您的系统已安装 Python 3.13.3，可以正常使用。

### 2. 安装项目依赖

打开终端（Terminal），执行以下步骤：

#### 步骤1：进入项目目录
```bash
cd "/Users/kiukiu/Five Elements Test"
```

#### 步骤2：安装Python依赖包
```bash
pip3 install -r requirements.txt
```

或者如果pip3不可用，尝试：
```bash
python3 -m pip install -r requirements.txt
```

**依赖包包括：**
- Flask (Web框架)
- flask-cors (跨域支持)
- openai (AI分析功能，可选)

### 3. （可选）配置OpenAI API Key

如果您想使用AI智能分析功能，需要设置API Key：

#### 方法1：临时设置（当前终端会话有效）
```bash
export OPENAI_API_KEY="your-api-key-here"
```

#### 方法2：永久设置（推荐）
编辑您的shell配置文件（~/.zshrc 或 ~/.bash_profile）：
```bash
echo 'export OPENAI_API_KEY="your-api-key-here"' >> ~/.zshrc
source ~/.zshrc
```

**注意：** 如果不设置API Key，网站仍然可以运行，但只会提供基础分析结果，不会有详细的AI分析。

## 🌍 访问网站

### 启动成功后的提示

当您看到类似以下信息时，说明网站已成功启动：
```
 * Running on http://127.0.0.1:5000
 * Running on http://0.0.0.0:5000
```

### 访问方式

#### 方式1：本地访问（推荐）
1. 打开您的浏览器（Chrome、Safari、Firefox等）
2. 在地址栏输入以下任一地址：
   - `http://localhost:5000`
   - `http://127.0.0.1:5000`
3. 按回车键访问

#### 方式2：同一网络下的其他设备访问
如果您的手机或电脑和运行服务器的电脑在同一WiFi网络下：
1. 找到运行服务器的电脑的IP地址（在终端输入 `ifconfig` 或 `ipconfig`）
2. 在其他设备的浏览器中输入：`http://[服务器IP地址]:5000`
   - 例如：`http://192.168.1.100:5000`

## 📱 使用网站

### 操作步骤

1. **选择出生日期**
   - 点击"出生日期"输入框
   - 选择您的出生年月日（必填）

2. **选择出生时间（可选）**
   - 如果知道具体出生时间，点击"出生时间"输入框选择
   - 如果不知道，可以留空不填

3. **开始分析**
   - 点击"开始分析"按钮
   - 等待几秒钟，系统会计算八字并进行分析

4. **查看结果**
   - 八字信息：显示年柱、月柱、日柱、时柱
   - 日主（日柱）：显示您的日主信息
   - 五行分布：显示五行分布情况
   - 详细分析：包含AI分析的喜用神等信息

## ⚠️ 常见问题

### 问题1：端口被占用
**错误信息：** `Address already in use`

**解决方法：**
- 修改 `app.py` 文件中的端口号（默认5000）
- 或者关闭占用5000端口的其他程序

### 问题2：依赖安装失败
**解决方法：**
- 尝试使用 `pip3 install --user -r requirements.txt`
- 或者使用虚拟环境：
  ```bash
  python3 -m venv venv
  source venv/bin/activate
  pip install -r requirements.txt
  ```

### 问题3：无法访问网站
**检查清单：**
- ✅ 确认服务器已成功启动（看到"Running on..."信息）
- ✅ 确认浏览器地址栏输入正确
- ✅ 确认防火墙没有阻止5000端口
- ✅ 尝试使用 `http://127.0.0.1:5000` 而不是 `localhost`

### 问题4：AI分析不工作
**可能原因：**
- 未设置 OPENAI_API_KEY
- API Key 无效或过期
- 网络连接问题

**解决方法：**
- 检查是否设置了环境变量：`echo $OPENAI_API_KEY`
- 重新设置正确的API Key
- 即使没有API Key，网站仍可正常使用，只是分析结果为基础版本
