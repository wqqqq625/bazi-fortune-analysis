# 📦 项目移动说明

## ✅ 可以安全移动

您的项目**可以移动到任何位置**，因为：
- ✅ 所有代码都使用相对路径
- ✅ 没有硬编码的绝对路径
- ✅ 所有文件都在同一个文件夹内

## 🚀 移动方法

### 方法1：使用 Finder（推荐，最简单）

1. **打开 Finder**
2. **找到项目文件夹**：`Five Elements Test`
3. **拖拽到新位置**（例如：桌面、Documents文件夹等）
4. **完成！** 移动后项目即可在新位置使用

### 方法2：使用终端命令

#### 移动到桌面
```bash
mv "/Users/kiukiu/Five Elements Test" ~/Desktop/
```

#### 移动到文档文件夹
```bash
mv "/Users/kiukiu/Five Elements Test" ~/Documents/
```

#### 移动到自定义位置
```bash
mv "/Users/kiukiu/Five Elements Test" /您想要的路径/
```

## 📝 移动后的使用

移动完成后，只需：

1. **打开终端**
2. **进入新位置的项目文件夹**
   ```bash
   cd ~/Desktop/"Five Elements Test"  # 如果移动到桌面
   # 或者
   cd ~/Documents/"Five Elements Test"  # 如果移动到文档
   ```
3. **启动网站**（和之前一样）
   ```bash
   python3 app.py
   ```

## ⚠️ 注意事项

1. **确保移动完整**：移动时确保整个 `Five Elements Test` 文件夹一起移动
2. **不要移动单个文件**：所有文件需要保持在同一文件夹内
3. **路径中的空格**：如果路径包含空格，记得用引号括起来

## 🔄 如果移动后遇到问题

如果移动后无法运行，检查：
- ✅ 所有文件是否都在新文件夹中
- ✅ 是否在新位置打开终端
- ✅ Python依赖是否已安装（如果没安装，重新运行 `pip3 install -r requirements.txt`）

## 💡 推荐位置

- **桌面** (`~/Desktop/`) - 方便访问
- **文档** (`~/Documents/`) - 更整洁
- **项目文件夹** (`~/Projects/` 或 `~/Code/`) - 如果有很多项目

---

**总结：移动很简单，拖拽即可！移动后使用方法完全一样。**
