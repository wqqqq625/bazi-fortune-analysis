# 🚀 启动网站指南

## 第一步：打开终端

在 Mac 上，按 `Command + 空格键`，输入 "终端" 或 "Terminal"，然后按回车。

## 第二步：进入项目目录

在终端中输入以下命令（复制粘贴即可）：

```bash
cd "/Users/kiukiu/Five Elements Test"
```

按回车执行。

## 第三步：安装依赖（如果还没安装）

输入以下命令安装 Python 依赖包：

```bash
pip3 install Flask flask-cors openai
```

或者：

```bash
python3 -m pip install Flask flask-cors openai
```

**注意**：如果提示权限错误，可以尝试：

```bash
pip3 install --user Flask flask-cors openai
```

## 第四步：启动服务器

输入以下命令启动服务器：

```bash
python3 app.py
```

按回车后，你会看到类似这样的输出：

```
 * Serving Flask app 'app'
 * Debug mode: on
 * Running on http://0.0.0.0:5001
```

**看到 "Running on http://0.0.0.0:5001" 就说明启动成功了！**

## 第五步：在浏览器中访问

1. 打开浏览器（Safari、Chrome、Firefox 都可以）
2. 在地址栏输入：`http://localhost:5001`
3. 按回车键

**现在你应该能看到网站的首页了！**

---

## 📋 访问方式总结

### 方式1：通过 Flask 服务器访问（推荐）
- 启动服务器：`python3 app.py`
- 访问地址：`http://localhost:5001`

### 方式2：直接打开 HTML 文件
- 在 Finder 中找到 `index.html` 文件
- 双击打开（会用浏览器打开）
- **注意**：这种方式前端能正常显示，但 API 调用需要服务器运行

---

## ⚠️ 常见问题

### 问题1：端口被占用
如果看到 "Address already in use"，说明 5001 端口被占用。

**解决方法**：
1. 找到占用端口的程序并关闭它
2. 或者修改 `app.py` 文件最后一行的端口号（比如改成 5002）

### 问题2：依赖安装失败
如果 pip3 命令失败，尝试：

```bash
python3 -m pip install --user Flask flask-cors openai
```

### 问题3：服务器启动后没反应
- 确保看到 "Running on http://0.0.0.0:5001" 的提示
- 在浏览器中访问 `http://localhost:5001`，不要关闭终端窗口
- 如果浏览器显示无法连接，检查防火墙设置

---

## 🛑 停止服务器

在运行服务器的终端窗口中，按 `Ctrl + C` 即可停止服务器。

---

## 💡 提示

- **保持终端窗口打开**：服务器运行时，终端窗口必须保持打开状态
- **不要关闭终端**：如果关闭终端，服务器会停止运行
- **如果修改了代码**：服务器在 Debug 模式下会自动重新加载，刷新浏览器即可看到变化
